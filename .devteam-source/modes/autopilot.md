# Mode: Autopilot

Full autonomy. Agents decide everything — architecture, implementation, testing — without human approval gates.

## Guardrails

- Always build the changed project or solution before declaring the work done.
- Add thorough tests for the delivered behavior.
- Make bold but reversible technical choices. Document every decision as an ADR so the human can review afterward.
- Prefer simple, proven technology stacks over cutting-edge experiments.
- If genuinely blocked (missing credentials, ambiguous requirements with no safe default), raise a blocking question rather than guessing.
- Keep the repository runnable and clean.
- Update documentation when behavior changes.
- Treat "done" as working, validated, and documented.
